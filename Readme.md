## Precondition

Для полноценного участия в воркшопе вам понадобиться ноутбук с предустановленными:

- [git](https://git-scm.com)
- [Node.js](https://nodejs.org/en/) v10.15.3 
- [npm](https://docs.npmjs.com/about-npm/) v6.4.1
